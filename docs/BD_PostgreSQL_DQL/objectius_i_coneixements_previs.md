# ![](icon_objectives.gif) Objetivos y Conocimientos previos


El objetivo primordal es **dominar el lenguaje SQL**, para poder hacer
sentencias de dificultad media-alta. En particular se tendrán que saber hacer:

  * Consultas SELECT, para sacar información de las tablas. Concretamente deberá conocerse a la perfección todas sus cláusulas.
  * Consultas SELECT con combinaciones o reuniones de tablas, incluyendo las combinaciones externas.
  * Consultas de unión
  * Subconsultas
  * Consultas para crear, modificar y eliminar tablas y vistas.
  * Consultas para insertar, modificar y eliminar filas de una tabla.
